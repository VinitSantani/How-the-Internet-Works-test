About This Project

I, Vinit Santani, am a first semester full time graduate student for Business and Information Systems. I have completed my undergraduate degree from Mumbai, India.

The purpose of this assignment is to teach us how to create and develop a fully functioning website.

I did not have any previous coding experience until last time so naturally web development with html and bootstrap was a scary task for the first time, but after watching Professor Williams's demo videos and doing some additional research I was able to successfully complete the task at hand!

I got a chance to solidify the git collaboration skills from the previous project and am now more comfortable using PyCharm and utilizing some of is unique features for web development. Taking the time to learn about html and bootstrapping will pay dividends in the future as the internet becomes more and more prevalent. It is impossible to escape using such a core language as html and bootstrapping enhances the webpages that we develop.

In addition to learning how to use these languages, I also learned about the how the internet works. This gave a crucial background that helped put into context how we got to where we are today.
